## Install

- Install [Hack Nerd Font](https://www.nerdfonts.com/)
- Install lua, libluadev, luarock
- Install [ripgreg](https://github.com/BurntSushi/ripgrep)
- Install [Zsh Syntax Highlighting](https://github.com/catppuccin/zsh-syntax-highlighting)
- Install [Catppuccin Theme for Tilix](https://github.com/catppuccin/tilix)
